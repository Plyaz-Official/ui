# Calendar

`Calendar` is a date picker component built with react-day-picker. It provides a flexible and accessible way to select dates, date ranges, and multiple dates. Styled with Tailwind CSS and integrated with the design system.

The calendar component supports various selection modes, custom styling, and accessibility features.

## Features

- Multiple selection modes (`single`, `range`, `multiple`)
- Customizable appearance with Tailwind CSS
- Accessible with proper ARIA attributes
- Support for disabled dates and date ranges
- Flexible layout options
- Week number support

## Usage Guidelines

- Use `mode="single"` for selecting one date
- Use `mode="range"` for selecting date ranges
- Use `mode="multiple"` for selecting multiple dates
- Set `showOutsideDays={false}` to hide previous/next month days
- Use `captionLayout="dropdown"` for month/year dropdowns

## Do

- Use appropriate selection modes for your use case
- Provide clear visual feedback for selected dates
- Use disabled dates for unavailable dates
- Consider accessibility when customizing

## Don't

- Use complex date logic without proper testing
- Forget to handle edge cases in date selection
- Make the calendar too small to interact with
- Override accessibility features
