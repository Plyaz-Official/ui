# ExternalLink

`ExternalLink` is a styled and accessible anchor (`<a>`) component used to link to external resources. It enforces semantic HTML and security best practices and supports Tailwind utility classes for visual control.

## Features

- Renders as a native anchor tag (`<a>`)
- Automatically applies `target="_blank"` and `rel="noopener noreferrer"`
- Fully styleable with Tailwind CSS
- Accepts all native anchor element props (`href`, `aria-*`, `data-*`, etc.)

## Usage Guidelines

- Use for external URLs where new tabs are required.
- Always include proper security attributes for external navigation.
- Compose with Tailwind classes for consistent hover, focus, and typography.
- Use meaningful link text that clearly describes the destination.

## Do

- Use `ExternalLink` to navigate to external websites or tools
- Include `rel="noopener noreferrer"` when using `target="_blank"`
- Apply Tailwind classes for styling and state changes
- Ensure accessibility through proper text, contrast, and focus state

## Don’t

- Use `ExternalLink` to trigger in-app navigation (use `Link` from router instead)
- Forget accessibility — avoid vague link texts like “Click here”
- Use `target="_blank"` without `rel="noopener noreferrer"` — it's a security risk
