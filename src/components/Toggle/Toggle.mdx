# Toggle

`Toggle` is a two-state button that can be toggled on and off. It provides a way to create toggle controls with proper styling and accessibility. Built with Radix UI and styled with Tailwind CSS.

The toggle component provides a powerful way to create toggle controls with proper focus management and accessibility.

## Features

- Two-state toggle control
- Focus management and keyboard navigation
- Accessible with proper ARIA attributes
- Support for controlled and uncontrolled modes
- Customizable styling and behavior
- Multiple variants and sizes
- Icon and text support

## Usage Guidelines

- Use toggles for binary state controls
- Keep labels concise and descriptive
- Use appropriate variants for context
- Consider accessibility requirements
- Use consistent sizing

## Do

- Use toggles for binary state controls
- Provide clear labels and descriptions
- Use appropriate variants
- Consider accessibility
- Use consistent sizing

## Don't

- Use toggles for complex state management
- Make toggles too small to interact with
- Forget to provide labels
- Overuse in your interface
