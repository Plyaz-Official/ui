# Tooltip

A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it.

## Features

- **Multiple positioning options**: `top`, `right`, `bottom`, `left`
- **Accessibility**: Built with Radix UI primitives for full accessibility support
- **Customizable**: Supports custom styling and theming
- **Keyboard Navigation**: Full keyboard support for interactions
- **Touch Support**: Optimized for touch devices

## Styling

The `Tooltip` component uses Tailwind CSS classes and can be customized with additional classes:

```tsx
<TooltipContent className="bg-blue-500 text-white" />
<TooltipContent side="bottom" align="start" />
```

## Accessibility

The `Tooltip` component provides accessibility support:

- **Screen Reader Support**: Proper ARIA attributes and roles
- **Keyboard Navigation**: Full keyboard support for interactions
- **Focus Management**: Proper focus handling and indicators
- **Touch Support**: Optimized for touch devices

## Best Practices

1. **Use meaningful content**: Provide clear and descriptive tooltip content
2. **Consider positioning**: Choose appropriate side based on available space
3. **Avoid overuse**: Don't use tooltips for information that should be visible
4. **Test with keyboard**: Ensure keyboard navigation works properly
5. **Keep content concise**: Tooltips should be brief and to the point
