# Tooltip

`Tooltip` is a component used for displaying contextual information when users hover over or focus on an element. It supports multiple positioning options and can be styled using Tailwind CSS utility classes.

ShadCN's Tooltip is built on top of the Radix UI tooltip behavior and integrates seamlessly with Tailwind for customization and responsiveness.

## Features

- Multiple positioning options (`top`, `right`, `bottom`, `left`)
- Supports alignment (`start`, `center`, `end`)
- Keyboard accessible by default
- Fully customizable with Tailwind CSS

## Usage Guidelines

- Use `side` to define the position of the tooltip.
- Combine with icon components from `lucide-react` if needed.
- Prefer descriptive content for accessibility.
- Avoid nesting interactive elements inside tooltips.

## Do

- Use `side="top"` for buttons in top navigation
- Use `side="right"` for buttons in left sidebar
- Apply Tailwind classes when you need layout/styling tweaks

## Don't

- Overuse tooltips in complex UI
- Nest interactive elements inside tooltips
- Forget to provide meaningful content
