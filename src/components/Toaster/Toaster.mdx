# Toaster

The `Toaster` component provides toast notifications with theme support. It is built with Sonner and styled with Tailwind CSS, offering a flexible and accessible toast solution.

## Features

- **Toast Notifications**: Provides toast notifications for user feedback
- **Theme Support**: Automatically adapts to light/dark themes
- **Accessibility**: Built with Sonner for full accessibility support
- **Customizable**: Supports custom styling and theming
- **Responsive**: Adapts to different screen sizes and orientations
- **Keyboard Navigation**: Full keyboard support for navigation
- **Touch Support**: Optimized for touch devices

## Props

| Prop          | Type                                                                                              | Default          | Description                                    |
| ------------- | ------------------------------------------------------------------------------------------------- | ---------------- | ---------------------------------------------- |
| `theme`       | `"light" \| "dark" \| "system"`                                                                   | `"system"`       | The theme of the toaster                       |
| `position`    | `"top-left" \| "top-center" \| "top-right" \| "bottom-left" \| "bottom-center" \| "bottom-right"` | `"bottom-right"` | The position of the toaster                    |
| `expand`      | `boolean`                                                                                         | `false`          | Whether to expand the toaster                  |
| `richColors`  | `boolean`                                                                                         | `false`          | Whether to use rich colors                     |
| `closeButton` | `boolean`                                                                                         | `false`          | Whether to show close button                   |
| `className`   | `string`                                                                                          | -                | Additional CSS classes to apply to the toaster |
| `...props`    | `ToasterProps`                                                                                    | -                | Additional props passed to the Sonner Toaster  |

## Styling

The `Toaster` component uses Tailwind CSS classes and can be customized with additional classes:

```tsx
<Toaster className='custom-toaster' />
```

## Accessibility

The `Toaster` component is built with Sonner and provides:

- **Keyboard Navigation**: Full keyboard support for navigation
- **Screen Reader Support**: Proper ARIA attributes and roles
- **Focus Management**: Focus indicators and proper focus handling
- **Touch Support**: Optimized for touch devices
- **Announcements**: Announces toast notifications to screen readers

## Performance

The `Toaster` component is optimized for performance:

- **Efficient Rendering**: Only re-renders when necessary
- **Smooth Animations**: Provides smooth toast animations
- **Memory Efficient**: Minimal memory usage
- **Responsive**: Adapts to different screen sizes and orientations

## Best Practices

1. **Use appropriate toast types**: Choose the right toast type for your message
2. **Provide meaningful content**: Ensure toast content is clear and actionable
3. **Handle user interactions**: Use actions for interactive toasts
4. **Test with keyboard**: Ensure keyboard navigation works properly
5. **Consider mobile**: Test on mobile devices for touch interactions
6. **Use controlled state**: Use controlled state for complex interactions
7. **Provide visual feedback**: Show clear visual feedback for user actions
