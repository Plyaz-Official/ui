# Toaster

An opinionated toast component for React.

## Features

- **Toast Notifications**: Provides toast notifications with theme support
- **Action Support**: Includes action buttons for user interaction
- **Theme Integration**: Supports light and dark themes
- **Customizable**: Supports custom styling and positioning
- **Accessible**: Built with accessibility in mind

```tsx

```

## Accessibility

The `Toaster` component provides accessibility support:

- **Screen Reader Support**: Proper ARIA attributes and roles
- **Keyboard Navigation**: Full keyboard support for interactions
- **Focus Management**: Proper focus handling for toast actions
- **Announcements**: Announces toast messages to screen readers

## Best Practices

1. **Use meaningful messages**: Provide clear and concise toast messages
2. **Include actions when appropriate**: Add action buttons for important toasts
3. **Don't overuse**: Avoid showing too many toasts at once
4. **Provide context**: Include descriptions when helpful
5. **Test with screen readers**: Ensure accessibility works properly
