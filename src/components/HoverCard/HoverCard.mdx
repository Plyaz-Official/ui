# HoverCard

`HoverCard` is a card that appears when a user hovers over a trigger element. It provides a way to display additional information or content without taking up permanent space. Built with Radix UI and styled with Tailwind CSS.

The hover card component provides a powerful way to create contextual information displays with proper hover management and accessibility.

## Features

- Contextual information display on hover
- Focus management and keyboard navigation
- Accessible with proper ARIA attributes
- Support for controlled and uncontrolled modes
- Customizable delays and alignments
- Responsive design with proper animations

## Usage Guidelines

- Use hover cards for additional information or context
- Keep content concise and relevant
- Provide clear and useful information
- Use appropriate delays for better UX
- Consider mobile touch interactions

## Do

- Use hover cards for contextual information
- Keep content concise and relevant
- Use appropriate delays
- Consider accessibility
- Provide clear information

## Don't

- Use hover cards for critical information
- Make content too complex or long
- Forget to handle mobile interactions
- Overuse hover cards in your interface
