# Link

The `Link` component is a styled anchor element used for navigation.

---

## Usage Guidelines

- Use for any inline navigation, both internal and external.
- Use semantic anchor tags (`<a>`) for proper SEO and accessibility.
- Add `target="_blank"` and `rel="noopener noreferrer"` for external links.
- Combine with utility classes to match the desired visual style (underline, hover states, etc.).
- Avoid using `Link` on buttons or non-interactive elements.

---

## Best Practices

- Use meaningful link text that clearly describes the destination.
- Keep accessibility in mind by ensuring sufficient color contrast.
- Ensure links work properly with keyboard navigation and screen readers.
- Test link interactions and hover/focus states in Storybook using the accessibility and interaction addons.
