# ContextMenu

`ContextMenu` is a right-click menu that appears when users interact with elements. It provides a contextual set of actions and options. Built with Radix UI and styled with Tailwind CSS.

The context menu component provides a powerful way to add contextual actions to any element with proper keyboard navigation and accessibility.

## Features

- Right-click activation
- Keyboard navigation support
- Accessible with proper ARIA attributes
- Support for sub-menus and groups
- Checkbox and radio items
- Customizable styling and behavior

## Usage Guidelines

- Use context menus for contextual actions
- Keep menu items organized and logical
- Use separators to group related items
- Provide clear, descriptive labels
- Consider keyboard accessibility

## Do

- Use context menus for contextual actions
- Keep menu items organized and logical
- Use separators to group related items
- Provide clear, descriptive labels

## Don't

- Use context menus for primary navigation
- Make menu items too long or complex
- Forget to provide keyboard alternatives
- Overuse context menus in simple interfaces
