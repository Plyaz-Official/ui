# RadioGroup

The `RadioGroup` component is a set of radio buttons that allows users to select a single option from multiple choices. It provides a way to create mutually exclusive selections. Built with Radix UI and styled with Tailwind CSS.

## Features

- **Mutually Exclusive Selection**: Only one option can be selected at a time
- **Keyboard Navigation**: Full keyboard support for accessibility
- **Customizable Styling**: Flexible styling with Tailwind CSS classes
- **Form Integration**: Works seamlessly with forms and validation
- **Accessibility**: Built with ARIA attributes and screen reader support

## Usage

### Basic Usage

<Canvas of={RadioGroupStories.Default} />

```tsx
import { RadioGroup, RadioGroupItem, Label } from '@/components';

<RadioGroup defaultValue='option1'>
  <div className='flex items-center space-x-2'>
    <RadioGroupItem value='option1' id='option1' />
    <Label htmlFor='option1'>Option 1</Label>
  </div>
  <div className='flex items-center space-x-2'>
    <RadioGroupItem value='option2' id='option2' />
    <Label htmlFor='option2'>Option 2</Label>
  </div>
</RadioGroup>;
```

## Accessibility

The RadioGroup component follows WAI-ARIA guidelines:

- Uses `role="radiogroup"` for the container
- Uses `role="radio"` for individual items
- Provides proper keyboard navigation
- Supports screen readers with appropriate ARIA attributes
- Maintains focus management

## Styling

The component uses Tailwind CSS for styling and can be customized with additional classes:

```tsx
<RadioGroup className='grid gap-2 grid-cols-[1rem_1fr] items-center'>
  <RadioGroupItem value='option1' id='option1' />
  <Label htmlFor='option1'>Option 1</Label>
</RadioGroup>
```
