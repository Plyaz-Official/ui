# Table

Powerful table and datagrids built using TanStack Table.

## Features

- **Structured Data Display**: Clean way to display tabular data
- **Accessibility**: Built with semantic HTML and ARIA attributes
- **Customizable**: Supports custom styling and theming
- **Responsive**: Adapts to different screen sizes with horizontal scrolling
- **Keyboard Navigation**: Full keyboard support
- **Touch Support**: Optimized for touch devices


## Accessibility

The `Table` component provides accessibility support:

- **Semantic HTML**: Uses proper table elements for screen readers
- **ARIA Support**: Includes proper ARIA attributes
- **Keyboard Navigation**: Full keyboard support for table interactions
- **Screen Reader Support**: Proper table structure for assistive technologies

## Best Practices

1. **Use proper table structure**: Always include TableHeader, TableBody, and proper table elements
2. **Provide captions**: Use TableCaption to describe the table content
3. **Use semantic headers**: Use TableHead for column headers
4. **Handle responsive design**: Consider horizontal scrolling for large tables
5. **Test with screen readers**: Ensure accessibility works properly
