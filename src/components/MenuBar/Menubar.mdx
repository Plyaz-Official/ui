# Menubar

`Menubar` is a headless, accessible horizontal menu bar built with Radix UI. It is wrapped by ShadCN UI and styled using Tailwind. Use it to group top-level actions with dropdowns for navigation or commands. Supports keyboard navigation, accessibility, and custom styling.

## Features

- Built on top of Radix UI
- Fully accessible with keyboard and screen reader support
- Portal support for dropdown rendering
- Composable: combine `MenubarMenu`, `MenubarTrigger`, `MenubarContent`, `MenubarItem`, and more
- Styled using Tailwind CSS utility classes

## Usage Guidelines

### Do

- Use for top-level app actions like File, Edit, View
- Customize `MenubarItem` with shortcuts using `MenubarShortcut`
- Use `MenubarSeparator` to visually group items
- Place `MenubarContent` inside `MenubarMenu` to ensure correct behavior

### Donâ€™t

- Use for nested, deep menu structures (use `DropdownMenu` instead)
- Place `MenubarContent` outside the `MenubarMenu` hierarchy
- Use raw click handlers to detect open state (use controlled selection instead)
