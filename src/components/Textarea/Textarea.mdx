# Textarea

`Textarea` is a form control that allows users to enter multi-line text. It provides a way to input longer text content with proper styling and accessibility. Built with Radix UI and styled with Tailwind CSS.

The textarea component provides a powerful way to create multi-line text inputs with proper focus management and accessibility.

## Features

- Multi-line text input
- Focus management and keyboard navigation
- Accessible with proper ARIA attributes
- Support for controlled and uncontrolled modes
- Customizable styling and behavior
- Resize functionality
- Character limit support

## Usage Guidelines

- Use textareas for multi-line text input
- Keep labels concise and descriptive
- Use appropriate sizing for content
- Consider character limits for user guidance
- Provide clear validation feedback

## Do

- Use textareas for multi-line text input
- Provide clear labels and descriptions
- Use appropriate sizing
- Consider character limits
- Provide validation feedback

## Don't

- Use textareas for single-line input (use input)
- Make textareas too small for content
- Forget to provide labels
- Overuse in your interface
