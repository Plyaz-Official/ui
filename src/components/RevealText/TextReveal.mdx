# Textarea

Displays a form textarea or a component that looks like a textarea.

## Features

- **Multi-line Input**: Supports multi-line text input
- **Accessibility**: Built with proper ARIA attributes and keyboard support
- **Customizable**: Supports custom styling and theming
- **Responsive**: Adapts to different screen sizes
- **Form Integration**: Works seamlessly with form components

## Styling

The `Textarea` component uses Tailwind CSS classes and can be customized with additional classes:

```tsx
<Textarea className="min-h-[100px]" />
<Textarea className="border-blue-500 focus-visible:border-blue-500" />
<Textarea className="resize-none" />
```

## Accessibility

The `Textarea` component provides accessibility support:

- **Screen Reader Support**: Proper ARIA attributes and roles
- **Keyboard Navigation**: Full keyboard support for input
- **Focus Management**: Proper focus handling and indicators
- **Label Association**: Proper label association for screen readers

## Best Practices

1. **Use labels**: Always provide clear labels for textareas
2. **Provide placeholders**: Use placeholder text to guide users
3. **Handle validation**: Provide clear error messages and validation feedback
4. **Consider sizing**: Use appropriate rows and resize options
5. **Test with keyboard**: Ensure keyboard navigation works properly
