# Design Tokens

To ensure visual consistency across the system, our components rely on standardized Tailwind CSS utility classes.
These tokens are not custom CSS variables, but structured utility-first values for color, spacing, and typography — aligned with Tailwind’s design system.

---

### Custom Colors (if we need )

- **Primary** — Brand accent color → `bg-primary`
- **Muted** — Background neutral → `bg-muted`
- **Text Primary** — Main text color → `text-primary`
- **Text Muted** — Secondary text color → `text-muted`

---

### Font Weight

- **font-normal** — 400
- **font-medium** — 500
- **space-semibold** — 600
- **space-bold** — 700
